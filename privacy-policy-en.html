Privacy Policy – Diportista

Last updated:

1) Data processed

Diportista may store on-device user-entered data and, if authorized, location data to track an outing.

2) Where data is stored

Data is stored locally on your device. The app does not automatically send data to external servers.

3) Permissions

  • Location: used to record GPS track during an outing, if enabled.
  • Photos/Videos: used only as outing attachments, if selected by the user.

4) Sharing

When the user uses “Share”, the selected content is shared through apps installed on the device.

5) Backup & restore

If the user exports/imports backups, the user is responsible for protecting those files.

For questions or requests, contact: diportista.app@gmail.com
