Privacy Policy – Diportista

Ultimo aggiornamento:

1) Dati trattati

Diportista può salvare localmente sul dispositivo dati inseriti dall’utente (uscite, consumi, spese) e, se autorizzato, dati di posizione per tracciare l’uscita.

2) Dove sono salvati i dati

I dati sono archiviati in locale sul dispositivo. L’app non invia automaticamente i dati a server esterni.

3) Permessi

  • Posizione: usata per registrare la traccia GPS durante l’uscita, se attivata.
  • Foto/Video: usati solo come allegati alle uscite, se scelti dall’utente.

4) Condivisione

Quando l’utente usa “Condividi”, i contenuti selezionati (testo/media) vengono condivisi tramite le app presenti sul dispositivo.

5) Backup e ripristino

Se l’utente esporta o importa backup, è responsabile della protezione dei file (che possono contenere dati personali).

Per domande o richieste, puoi contattarci a: diportista.app@gmail.com
